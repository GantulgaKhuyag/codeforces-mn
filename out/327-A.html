Урвуулах тоглоом

Иахуб уйдсандаа цаасан дээр тоглодог тоглоом зохиожээ.

Эхлээд n ширхэг a1, a2, ... ,an тоо бичнэ. Бичсэн тоонууд нь зөвхөн 0 юм уу 1 байна. Дараа нь дурын i, j (1 ≤ i ≤ j ≤ n) хоёр дугаар сонгож аваад [i, j] завсар дахь бүх ak-гийн (i ≤ k ≤ j) утгыг урвуулна. (x = 1 − x үйлдлийг хийнэ гэсэн үг)

Тоглоомын зорилго бол зөвхөн нэг үйлдэл хийгээд боломжит хамгийн олон нэгүүдийг гаргаж авах юм. Иахубын тоглоомыг шийддэг програм зохионо уу.

Оролт

Эхний мөрөнд n тоо (1 ≤ n ≤ 100).

Хоёр дахь мөрөнд n ширхэг a1, a2, ..., an тоонууд байрлана. Мэдээж энэ тоонууд нь зөвхөн 0 юм уу 1 байна.

Гаралт

Нэг нүүдлийн дараа гаргаж авч чадах 1-үүдийн хамгийн их тоо.
