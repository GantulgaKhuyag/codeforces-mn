Агуу Мөөгөн Хааны Айлчлал

Агуу Мөөгөн Хаан одойчууд дээр айлчлан очсон боловч бүх одойчууд бус зөвхөн цөөн хэдэн сонгогдсон одойчууд л хаанд бараалхах эрхтэй болно.

ХБЕХ(k2l + 1, k2l + 1 + 1, . . . , k2r + 1) тооны одойчууд хаанд бараалхах эрхтэй ба k, l, r тоонуудыг хаан өөрөө өгдөг болно.

Одой түүхчид Агуу Мөөгөн Хааны айлчлал бүрийг баримтжуулан тэмдэглэглэхээр шийдсэн ба тэдэнд айлчлал бүрийн k, l, r тоонууд ба pi анхны тоо байгаа бол айлчлал бүрд хаанд бараалхах эрхтэй одойчуудын тоог pi-д хуваасан үлдэгдлийг олж өгнө үү.

Оролт

Эхний мөрөнд хааны нийт айлчлалын тоо болох t(1 <  = t <  = 105) байрлах ба дараагийн t мөрөнд хоорондоо хоосон зайгаар тусгаарлагдсан бүхэл тоонууд ki, li, ri ба pi(1 <  = ki <  = 106, 0 <  = l <  = r <  = 1018, 2 <  = pi <  = 109)-үүд байх ба pi болгон нь заавал анхны тоо байна.

Гаралт

Мөр болгонд хаанд бараалхах эрхтэй одойчуудын тоог анхны тоо pi-д хуваасан үлдэгдлийг хэвлэ. Айчлалуудын хариунуудыг оролтонд орж ирсэн дарааллаар нь хэвлэнэ.
