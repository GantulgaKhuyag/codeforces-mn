Даалуун цуваа

Вася даалуу өрөх дуртай. Тэр нийтлэг даалуунаас залхан ялгаатай өндөртэй даалуунуудыг ашигладаг байв. Тэр n даалууг нэг тэнхлэгийн дагуу зүүнээс баруун тийш өрөв. Даалуу бүр тэнхлэгтэй перпендикуляр бөгөөд тэнхлэг түүний суурийн төвийг дайрч өнгөрсөн байхаар байрлана. i-р домино xi координаттай, hi өндөртэй байна. Вася даалуу бүрийн хувьд түүнийг баруун тийш унагавал нийт хэдэн даалуу унахыг мэдэхийг хүсэв. Түүнд үүнийг хийхэд тусална уу.

Түүний сууринд хүрсэн тохиолдолд уг даалуу унана гэж үзнэ. Өөрөөр хэлбэл унаж байгаа даалуу [x + 1, x + h − 1] хэрчим дээрх бүх даалуунуудыг унагана.

Оролт

Эхний мөр даалуунуудын тоо болох n (1 ≤ n ≤ 105) бүхэл тоог агуулна. Дараагийн n мөрөнд даалуу бүрийн координат, өндрийг илэрхийлэх xi, hi ( − 108 ≤ xi ≤ 108, 2 ≤ hi ≤ 108) хоёр бүхэл тоо өгөгднө. Нэг цэг дээр хоёр даалуу байрлахгүй.

Гаралт

i-р даалууг баруун тийш унагахад нийт хэдэн даалуу (өөрийг нь оруулаад) унахыг илэрхийлэх n ширхэг zi бүхэл тоог зайгаар тусгаарлан хэвлэ.
