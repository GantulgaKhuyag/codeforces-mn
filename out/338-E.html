Манао дараах бодлогыг бодож байлаа:

Түүний бодолт зөв байсан боловч ажиллагаа нь хэтэрхий удаан байлаа. Танд түүний бодолтын хуурмаг-код өгөгджээ. (getAnswer функцын үр дүн нь бодлогын хариу)

getAnswer(a[1..n], b[1..len], h)
  answer = 0
  for i = 1 to n-len+1
    answer = answer + f(a[i..i+len-1], b, h, 1)
  return answer

f(s[1..len], b[1..len], h, index)
  if index = len+1 then
    return 1
  for i = 1 to len
    if s[index] + b[i] >= h
      mem = b[i]
      b[i] = 0
      res = f(s, b, h, index + 1)
      b[i] = mem
      if res > 0
        return 1
  return 0

Манаод алгоритмаа сайжруулахад нь тусална уу.

Оролт

Эхний мөрөнд n, len, h (1 ≤ len ≤ n ≤ 150000; 1 ≤ h ≤ 109) тоонууд зайгаар тусгаарлагдан өгөгдөнө.

Хоёр дахь мөрөнд len ширхэг зайгаар тусгаарлагдсан b1, b2, ..., blen (1 ≤ bi ≤ 109) тоонууд.

Гурав дахь мөрөнд n ширхэг зайгаар тусгаарлагдсан тоонууд a1, a2, ... , an (1 ≤ ai ≤ 109) байрлана.
