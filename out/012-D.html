Бөмбөг

Товч ѳгүүлбэр: N хатагтай бүрийн царайлаг, ухаалаг, баяныг илэрхийлэх Bi, Ii, Ri, i = 1, 2, . . . , N тоонууд ѳгѳгдѳв.

i-р хатагтайн хувьд Bi < Bj, Ii < Ij, Ri < Rj байх j гэсэн хатагтай байх юм бол i-р хатагтайг амиа хорлогч гэе. Тэгвэл амиа хорлогчдын тоог ол.

Оролт

Эхний мѳрѳнд бүхэл тоо N, (1 ≤ N ≤ 500000). Хоёрдахь мѳрѳнд N ширхэг бүхэл тоо Bi зайгаар тусгаарлагдан ѳгѳгднѳ. Гурав, дѳрѳвдэх мѳрѳнд үүнтэй адилаар Ii, Ri тоонууд ѳгѳгднѳ. 0 ≤ Bi, Ii, Ri ≤ 109 байна.
