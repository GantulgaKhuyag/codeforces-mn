Тавцангууд

Нэг хэмжээст ертѳнц n тавцантай. k-р тавцан (тавцангууд нэгээс эхлэн дугаарлагдна) [(k − 1)m, (k − 1)m + l], (l < m) хэрчмийг эзлэн байрлана. Царцаа 0 цэгээс эхлэн яг d нэгжээр үсэрнэ. Хэрвээ хоосон цэг таарвал доош унана. Тавцан дээр болон түүний ирмэгт буувал доош унахгүй. Доош унасан бол ахиж гарч ирэхгүй.

Оролт

Эхний мѳрѳнд тавцангийн тоо, царцааны харайлтын хэмжээ, тавцангийн байрлалыг илэрхийлэх n, d, m, l, (1 ≤ n, d, m, l ≤ 106, l < m) дѳрвѳн бүхэл тоо ѳгѳгднѳ.

Гаралт

Царцааны унах цэгийн координатыг тодорхойл.
